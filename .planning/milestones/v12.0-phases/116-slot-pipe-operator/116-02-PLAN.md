For `infer_slot_pipe` (new function), the logic for `x |N> f(a, b)`:
- Infer `lhs_ty` as before
- `slot = pipe.slot()` (1-indexed, N >= 2 by parse-time guarantee)
- zero-indexed insert position = `slot - 1`
- Count explicit args from `call.arg_list()` → `explicit_args: Vec<Ty>`
- Conflict check: if `(slot - 1) <= explicit_args.len()` → there are already args in that slot
  (explicit args fill positions 2..=explicit_args.len()+1, piped fills position N)
  i.e., if `slot - 1 <= explicit_args.len()` → conflict (position N is already occupied)
  Actually: explicit_args[0] is position 2, explicit_args[1] is position 3, etc.
  The piped value goes to position `slot` (1-indexed). Existing explicit args cover positions
  2..=(explicit_args.len() + 1). Conflict when `2 <= slot <= explicit_args.len() + 1`
  and the (slot-2)th explicit arg would be displaced.
  SIMPLER: after inserting piped at slot, explicit args shift. Think of it as:
    - Build the final arg list with piped at position (slot-1) in 0-indexed
    - The caller provided `explicit_args.len()` args for slots other than `slot`
    - Conflict = the caller already filled what would be inserted position
    - Per context decision: "Hard compile error when slot position N targets an argument
      position already explicitly provided". Example: `x |2> func(a, b)` where 2 explicit
      args are given, meaning positions 2 AND 3 are filled → piping to position 2 conflicts.
    - Rule: if `slot - 1 <= explicit_args.len()` → conflict (position `slot` is within
      the range of explicitly provided args; explicit args fill 2..N+1 sequentially starting
      from position 2). Wait — explicit args start at position 2 (position 1 is the piped).
    - Actually for `|>` (regular pipe): piped value → position 1. Explicit args → positions 2, 3, ...
    - For `|2>` (slot pipe): piped value → position 2. Explicit args → positions 1, 3, 4, ...
      No — re-read: `x |2> func(a, b)` means func(a, x, b)? Or func(x, a, b) shifted?
    - Per phase decision: "x |2> func(a, b) where position 2 is b → compiler rejects with conflict"
      So explicit args `(a, b)` occupy positions 1 and 2. Piping to position 2 = conflict with `b`.
    - So explicit args are treated as filling positions 1, 2, 3... and piped value INSERTS at slot.
    - Conflict: if `slot <= explicit_args.len()` (position slot is already filled by an explicit arg).
    - Insert: build `[explicit_args[0..slot-2], lhs_ty, explicit_args[slot-1..]]`
      (slot is 1-indexed, so slot-1 is 0-indexed insert position; explicit_args before insert point
      are [0..slot-2] i.e. first slot-2 args, then insert lhs, then rest)

fn lower_pipe_expr(&mut self, pipe: &PipeExpr) -> MirExpr {
    let lhs = pipe.lhs().map(|e| self.lower_expr(&e)).unwrap_or(MirExpr::Unit);
    let rhs = pipe.rhs();
    let ty = self.resolve_range(pipe.syntax().text_range());

    match rhs {
        Some(Expr::CallExpr(call)) => {
            let callee = call.callee().map(|e| self.lower_expr(&e));
            let mut args = vec![lhs];   // prepend lhs at position 0 (position 1)
            for arg in call.arg_list().args() { args.push(self.lower_expr(&arg)); }
            MirExpr::Call { func: Box::new(callee), args, ty }
        }
        Some(rhs_expr) => { /* bare fn: f(x) */ }
        None => MirExpr::Unit,
    }
}

fn lower_slot_pipe_expr(&mut self, pipe: &SlotPipeExpr) -> MirExpr {
    let lhs = pipe.lhs().map(|e| self.lower_expr(&e)).unwrap_or(MirExpr::Unit);
    let slot = pipe.slot().unwrap_or(2) as usize; // 1-indexed
    let insert_idx = slot - 1; // 0-indexed position to insert lhs
    let ty = self.resolve_range(pipe.syntax().text_range());

    match pipe.rhs() {
        Some(Expr::CallExpr(call)) => {
            let callee = call.callee().map(|e| self.lower_expr(&e));
            let mut explicit_args: Vec<MirExpr> = Vec::new();
            if let Some(arg_list) = call.arg_list() {
                for arg in arg_list.args() {
                    explicit_args.push(self.lower_expr(&arg));
                }
            }
            // Insert lhs at 0-indexed position (insert_idx), clamping to length
            let actual_idx = insert_idx.min(explicit_args.len());
            let mut args = explicit_args;
            args.insert(actual_idx, lhs);
            MirExpr::Call { func: Box::new(callee.unwrap()), args, ty }
        }
        Some(rhs_expr) => {
            // Bare function reference: treat like regular pipe (insert at first position)
            let func = self.lower_expr(&rhs_expr);
            MirExpr::Call { func: Box::new(func), args: vec![lhs], ty }
        }
        None => MirExpr::Unit,
    }
}

    ```rust
    /// Infer the type of a slot pipe expression: `lhs |N> rhs`
    fn infer_slot_pipe(
        ctx: &mut InferCtx,
        env: &mut TypeEnv,
        pipe: &SlotPipeExpr,
        types: &mut FxHashMap<TextRange, Ty>,
        type_registry: &TypeRegistry,
        trait_registry: &TraitRegistry,
        fn_constraints: &FxHashMap<String, FnConstraints>,
    ) -> Result<Ty, TypeError> {
        let lhs = pipe.lhs().ok_or_else(|| { /* error */ })?;
        let rhs = pipe.rhs().ok_or_else(|| { /* error */ })?;
        let slot = pipe.slot().unwrap_or(2); // 1-indexed, >= 2 by parse guarantee
        let insert_idx = (slot - 1) as usize; // 0-indexed insert position in the final arg list

        let lhs_ty = infer_expr(ctx, env, &lhs, types, type_registry, trait_registry, fn_constraints)?;
        let ret_var = ctx.fresh_var();

        match &rhs {
            Expr::CallExpr(call) => {
                let callee_expr = call.callee().ok_or_else(|| { /* error */ })?;
                let callee_ty = infer_expr(ctx, env, &callee_expr, types, type_registry, trait_registry, fn_constraints)?;

                // Infer explicit argument types
                let mut explicit_arg_types: Vec<Ty> = Vec::new();
                if let Some(arg_list) = call.arg_list() {
                    for arg in arg_list.args() {
                        let arg_ty = infer_expr(ctx, env, &arg, types, type_registry, trait_registry, fn_constraints)?;
                        explicit_arg_types.push(arg_ty);
                    }
                }

                // Conflict check: explicit args fill positions 1..=N (0-indexed: 0..N-1).
                // Piped value inserts at insert_idx (0-indexed). Conflict if insert_idx < explicit_arg_types.len().
                // Per design: `x |2> func(a, b)` — explicit args are (a, b) = positions 1 and 2.
                // Piping to slot 2 (insert_idx=1) conflicts with `b` (at index 1).
                if insert_idx < explicit_arg_types.len() {
                    let fn_name = if let Expr::NameRef(nr) = &callee_expr {
                        nr.text().unwrap_or("").to_string()
                    } else { String::new() };
                    let err = TypeError::SlotPositionConflict {
                        slot,
                        fn_name,
                        span: pipe.syntax().text_range(),
                    };
                    ctx.errors.push(err.clone());
                    return Err(err);
                }

                // Build the full arg type list with lhs inserted at insert_idx:
                // explicit_args[0..insert_idx] ++ [lhs_ty] ++ explicit_args[insert_idx..]
                // NOTE: since conflict check passed, insert_idx >= explicit_arg_types.len(),
                // so all explicit args come before the piped value.
                // i.e., full_args = explicit_arg_types ++ [lhs_ty] ++ padding-if-needed
                // Actually: insert at exact position means we need gap-filling if insert_idx > len.
                // Simple approach: pad with fresh type vars for any gap, then insert.
                let mut full_args: Vec<Ty> = Vec::new();
                for i in 0..insert_idx {
                    if i < explicit_arg_types.len() {
                        full_args.push(explicit_arg_types[i].clone());
                    } else {
                        full_args.push(ctx.fresh_var()); // gap-filling (unusual case)
                    }
                }
                full_args.push(lhs_ty.clone());
                // remaining explicit args after the insert point
                for i in insert_idx..explicit_arg_types.len() {
                    full_args.push(explicit_arg_types[i].clone());
                }

                // Arity-range check: if callee has a known Fun type, validate slot <= arity
                let resolved_callee = ctx.resolve(callee_ty.clone());
                if let Ty::Fun(ref params, _) = resolved_callee {
                    if (slot as usize) > params.len() {
                        let fn_name = if let Expr::NameRef(nr) = &callee_expr {
                            nr.text().unwrap_or("").to_string()
                        } else { String::new() };
                        let err = TypeError::SlotPipeOutOfRange {
                            slot,
                            fn_name,
                            arity: params.len(),
                            span: pipe.syntax().text_range(),
                        };
                        ctx.errors.push(err.clone());
                        return Err(err);
                    }
                }

                let expected_fn_ty = Ty::Fun(full_args.clone(), Box::new(ret_var.clone()));
                let origin = ConstraintOrigin::FnArg {
                    call_site: call.syntax().text_range(),
                    param_idx: insert_idx,
                };
                ctx.unify(callee_ty, expected_fn_ty, origin.clone())?;

                // Record type for the CallExpr node (mirrors infer_pipe)
                let resolved_call = ctx.resolve(ret_var.clone());
                types.insert(call.syntax().text_range(), resolved_call);

                // Where-clause constraint checking (mirrors infer_pipe's CallExpr arm)
                if let Expr::NameRef(name_ref) = &callee_expr {
                    if let Some(fn_name) = name_ref.text() {
                        if let Some(constraints) = fn_constraints.get(&fn_name) {
                            if !constraints.where_constraints.is_empty() {
                                let mut resolved_type_args: FxHashMap<String, Ty> = FxHashMap::default();
                                for (i, tp_name_opt) in constraints.param_type_param_names.iter().enumerate() {
                                    if let Some(tp_name) = tp_name_opt {
                                        if i < full_args.len() {
                                            resolved_type_args.insert(tp_name.clone(), ctx.resolve(full_args[i].clone()));
                                        }
                                    }
                                }
                                for (param_name, param_ty) in &constraints.type_params {
                                    if !resolved_type_args.contains_key(param_name) {
                                        resolved_type_args.insert(param_name.clone(), ctx.resolve(param_ty.clone()));
                                    }
                                }
                                let errors = trait_registry.check_where_constraints(&constraints.where_constraints, &resolved_type_args, origin.clone());
                                ctx.errors.extend(errors.clone());
                                if let Some(first_err) = errors.into_iter().next() {
                                    return Err(first_err);
                                }
                            }
                        }
                    }
                }
            }
            _ => {
                // Bare function reference: treat like regular pipe (insert lhs at position 0)
                let rhs_ty = infer_expr(ctx, env, &rhs, types, type_registry, trait_registry, fn_constraints)?;
                let expected_fn = Ty::Fun(vec![lhs_ty], Box::new(ret_var.clone()));
                ctx.unify(rhs_ty, expected_fn, ConstraintOrigin::Builtin)?;
            }
        }

        Ok(ret_var)
    }
    ```
