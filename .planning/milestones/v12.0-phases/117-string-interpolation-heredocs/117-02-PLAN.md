**Step 1: Add `trim_indent` function**

```rust
/// Apply Kotlin-style trimIndent to heredoc content.
///
/// Algorithm:
/// 1. Strip the leading newline that follows the opening `"""` (content starts after that newline).
/// 2. Find the indent of the closing `"""` by looking at the last non-empty line's leading spaces.
///    The closing `"""` is on its own indented line — the indent of the LAST line in the raw
///    content (before stripping) determines the trim level.
/// 3. Strip that many leading spaces/tabs from every line. Lines with fewer leading spaces than
///    the trim level are stripped only to their actual content start (no negative indent).
/// 4. Strip the trailing newline (the one that precedes the closing `"""`).
fn trim_indent(raw: &str) -> String {
    // Step 1: strip leading newline after opening """
    let content = if raw.starts_with('\n') {
        &raw[1..]
    } else if raw.starts_with("\r\n") {
        &raw[2..]
    } else {
        raw
    };

    // Step 2: find indent level from last line (closing """ line)
    let lines: Vec<&str> = content.split('\n').collect();
    let trim_level = if let Some(last) = lines.last() {
        last.chars().take_while(|c| *c == ' ' || *c == '\t').count()
    } else {
        0
    };

    // Step 3: strip trim_level chars from each line (except the last line which is the closing indent)
    // Step 4: drop the last line (it's just the closing """ indentation, not content)
    let content_lines = if lines.len() > 1 {
        &lines[..lines.len() - 1]
    } else {
        &lines[..]
    };

    let mut result = String::new();
    for (i, line) in content_lines.iter().enumerate() {
        let stripped = if line.len() >= trim_level {
            &line[trim_level..]
        } else {
            line.trim_start_matches(|c| c == ' ' || c == '\t')
        };
        if i > 0 {
            result.push('\n');
        }
        result.push_str(stripped);
    }
    result
}
```

**Step 2: Update `lower_string_expr` to detect triple-quoted strings and apply trimming**

For triple-quoted strings, the STRING_CONTENT tokens contain raw text including newlines and indentation. The trimming must be applied to the concatenated raw content BEFORE creating `MirExpr::StringLit`. However, since the content may be interspersed with INTERPOLATION segments, the trimming approach needs care.

For simplicity (Claude's discretion): apply `trim_indent` to each STRING_CONTENT segment individually by tracking whether we're in a triple-quoted context. The first STRING_CONTENT segment after STRING_START gets its leading newline stripped; the last STRING_CONTENT segment before STRING_END gets its trailing newline+indent stripped. Middle segments are stripped of their per-line indentation.

**Simpler correct approach:** Collect all STRING_CONTENT text (with their relative positions), then apply trim_indent to the assembled plain text, then split back. But this breaks for interpolations.

**Best approach for interpolation + trimming:**

1. Detect `is_triple` from STRING_START token.
2. If not triple, process as before (no trimming).
3. If triple: collect the full raw content by assembling STRING_CONTENT spans (no interpolation — concatenate the text). Then apply `trim_indent`. If there ARE interpolations, apply line-level stripping: for each STRING_CONTENT segment, strip `trim_level` leading chars from each line within it.

To determine `trim_level` for a triple-quoted string with interpolations: look at the last STRING_CONTENT segment (before STRING_END) — the last line of that segment gives the trim level.

Implement as follows in `lower_string_expr`:

```rust
fn lower_string_expr(&mut self, str_expr: &StringExpr) -> MirExpr {
    // Detect triple-quoted string from STRING_START token text
    let is_triple = str_expr.syntax().children_with_tokens()
        .find(|c| c.kind() == SyntaxKind::STRING_START)
        .and_then(|c| c.as_token())
        .map(|t| t.text().starts_with("\"\"\""))
        .unwrap_or(false);

    // For triple-quoted strings with no interpolation, collect all content and apply trim_indent.
    // For triple-quoted with interpolations, compute trim_level from last STRING_CONTENT and
    // apply strip_indent_lines() to each STRING_CONTENT segment.

    let trim_level: usize = if is_triple {
        // Find the last STRING_CONTENT token to determine closing indent
        str_expr.syntax().children_with_tokens()
            .filter(|c| c.kind() == SyntaxKind::STRING_CONTENT)
            .last()
            .and_then(|c| c.as_token())
            .map(|t| {
                let text = t.text();
                // The last line of the last STRING_CONTENT is the closing indent line
                text.lines().last().unwrap_or("").chars()
                    .take_while(|c| *c == ' ' || *c == '\t').count()
            })
            .unwrap_or(0)
    } else {
        0
    };

    let mut segments: Vec<MirExpr> = Vec::new();
    let mut is_first_content = is_triple; // first STRING_CONTENT needs leading newline stripped

    for child in str_expr.syntax().children_with_tokens() {
        match child.kind() {
            SyntaxKind::STRING_CONTENT => {
                let raw_text = child
                    .as_token()
                    .map(|t| unescape_string(t.text()))
                    .unwrap_or_default();

                let text = if is_triple {
                    apply_heredoc_content(raw_text, is_first_content, trim_level)
                } else {
                    raw_text
                };
                is_first_content = false;

                if !text.is_empty() {
                    segments.push(MirExpr::StringLit(text, MirType::String));
                }
            }
            SyntaxKind::INTERPOLATION => {
                is_first_content = false; // after first segment (which may be empty)
                if let Some(node) = child.as_node() {
                    for inner in node.children() {
                        if let Some(expr) = Expr::cast(inner) {
                            let typeck_ty = self.get_ty(expr.syntax().text_range()).cloned();
                            let lowered = self.lower_expr(&expr);
                            let converted = self.wrap_to_string(lowered, typeck_ty.as_ref());
                            segments.push(converted);
                        }
                    }
                }
            }
            _ => {}
        }
    }

    // existing concat logic unchanged...
    if segments.is_empty() {
        return MirExpr::StringLit(String::new(), MirType::String);
    }
    if segments.len() == 1 {
        return segments.pop().unwrap();
    }
    let mut result = segments.remove(0);
    for seg in segments {
        result = MirExpr::Call {
            func: Box::new(MirExpr::Var(
                "mesh_string_concat".to_string(),
                MirType::FnPtr(vec![MirType::String, MirType::String], Box::new(MirType::String)),
            )),
            args: vec![result, seg],
            ty: MirType::String,
        };
    }
    result
}
```

Add the helper `apply_heredoc_content`:

```rust
/// Process a STRING_CONTENT segment from a triple-quoted heredoc.
///
/// - `is_first`: strip the leading newline (the one after the opening `"""`)
/// - `trim_level`: strip this many leading spaces from each line
/// - For the last segment, the last line contains only the closing indent — it is dropped.
///   Detection: if the segment ends with a newline followed by trim_level spaces (and nothing
///   else after that), this is the closing indent line and must be stripped.
fn apply_heredoc_content(text: String, is_first: bool, trim_level: usize) -> String {
    // Strip leading newline from first segment
    let s = if is_first {
        if text.starts_with('\n') { text[1..].to_string() }
        else if text.starts_with("\r\n") { text[2..].to_string() }
        else { text }
    } else {
        text
    };

    // Check if this segment ends with a closing indent line (last line is all-whitespace)
    let is_last_segment = {
        let trimmed_end = s.trim_end_matches(|c| c == ' ' || c == '\t');
        trimmed_end.ends_with('\n') || trimmed_end.is_empty()
    };

    // Strip trim_level from each line, drop the final all-whitespace line if present
    let mut lines: Vec<&str> = s.split('\n').collect();

    // Drop last line if it's purely whitespace (closing indent line)
    if is_last_segment && lines.last().map(|l| l.trim().is_empty()).unwrap_or(false) {
        lines.pop();
    }

    let stripped_lines: Vec<String> = lines.iter().map(|line| {
        if line.len() >= trim_level {
            line[trim_level..].to_string()
        } else {
            line.trim_start_matches(|c: char| c == ' ' || c == '\t').to_string()
        }
    }).collect();

    stripped_lines.join("\n")
}
```
